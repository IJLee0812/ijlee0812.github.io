#백준 2869번 문제풀이_C

####출처
[LINK] (https://www.acmicpc.net/problem/2869)

####코드

<pre>
<code>
```C
int main(){
	int a, b, v, res;
	scanf("%d %d %d", &a, &b, &v);
	res = (v-b)/(a-b);
	if ((v-b)%(a-b) == 0)
		printf("%d\n", res);
	else
		printf("%d\n", res+1);
	return 0;
}
```
</code>
</pre>
####풀이

낮에 A미터 올라가고, 밤에 B미터 내려가므로 하루에 갈 수 있는 거리는
<strong>A-B 미터</strong> 가 됨.

또한 V미터를 완주한 날은 내려가는 B미터를 빼주면 총 이동거리가 나옴. 즉
<strong>V-B 미터</strong> 가 됨.

V-B가 A-B로 나누어 떨어지는 경우는 (V-B) / (A-B) 일이 걸리고
그렇지 않은 경우는 위 식에 1을 더해주어야 한다.
